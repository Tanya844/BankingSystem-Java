# üè¶ Java Banking System with MySQL and JavaFX

A simple Java project built by Tanya Pal for demonstrating JDBC, OOP, and JavaFX skills.

## Features
- Create new bank accounts
- Deposit and withdraw money
- Check balance and view transactions
- MySQL database integration
- JavaFX GUI interface

## How to Run
1. Add MySQL connector JAR to `lib/`
2. Compile:
   ```bash
   javac -cp '.:lib/mysql-connector-j-9.5.0.jar' src/*.java -d bin
   ```
3. Run:
   ```bash
   java -cp '.:lib/mysql-connector-j-9.5.0.jar:bin' MainFX
   ```
